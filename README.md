---

## 📚 Learning Roadmap

### JavaScript Topics
1. Variables (`let`, `const`, `var`)
2. Data Types (string, number, boolean, array, object)
3. Operators (arithmetic, comparison, logical)
4. Memory Concepts (stack vs heap, references)
5. Functions (declaration, arrow functions, parameters, return values)
6. Scope (global, local, block scope)
7. Control Flow (if/else, switch, loops: for, while, for...of)
8. DOM Manipulation (querySelector, innerHTML, styles)
9. Events (click, input, submit, event listeners)
10. ES6+ Features (template literals, destructuring, spread/rest, modules)
11. Asynchronous JavaScript (callbacks, Promises, async/await, fetch API)
12. Error Handling (try/catch, throwing errors)
